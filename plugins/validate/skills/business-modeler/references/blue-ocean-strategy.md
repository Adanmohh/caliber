## 1. RED OCEAN vs BLUE OCEAN — DIAGNOSTIC

Before applying Blue Ocean tools, diagnose the current competitive landscape.

### Red vs Blue Ocean Comparison

```
+---------------------------+---------------------------+
|      RED OCEAN            |      BLUE OCEAN           |
+---------------------------+---------------------------+
| Compete in existing       | Create uncontested        |
| market space              | market space              |
+---------------------------+---------------------------+
| Beat the competition      | Make competition          |
|                           | irrelevant                |
+---------------------------+---------------------------+
| Exploit existing demand   | Create and capture        |
|                           | new demand                |
+---------------------------+---------------------------+
| Make the value-cost       | Break the value-cost      |
| trade-off                 | trade-off                 |
+---------------------------+---------------------------+
| Align activities with     | Align activities with     |
| differentiation OR        | differentiation AND       |
| low cost                  | low cost simultaneously   |
+---------------------------+---------------------------+
```

### Red Ocean Diagnostic Checklist

Score each statement 1-5 (1=strongly disagree, 5=strongly agree):

| # | Indicator                                              | Score |
|---|--------------------------------------------------------|-------|
| 1 | Our industry competes primarily on price               |       |
| 2 | Products/services are becoming increasingly similar    |       |
| 3 | Market share gains come only at competitors' expense   |       |
| 4 | Industry growth is flat or declining                   |       |
| 5 | Marketing budgets are increasing but results declining |       |
| 6 | We define competitors narrowly (same industry only)    |       |
| 7 | Our strategy focuses on beating specific rivals        |       |
| 8 | Customer satisfaction is high but loyalty is low       |       |
| 9 | Innovation focuses on incremental improvements         |       |
|10 | Profit margins are shrinking industry-wide             |       |

**Scoring:**
- 40-50: Deep red ocean — urgent need for blue ocean shift
- 30-39: Reddening ocean — act within 12 months
- 20-29: Mixed — selective blue ocean opportunities exist
- 10-19: Relatively blue — maintain and extend

## 2. THE STRATEGY CANVAS

The Strategy Canvas is the core diagnostic and action framework of Blue Ocean
Strategy. It captures the current competitive landscape and visualizes your
strategic move.

### How to Build a Strategy Canvas

**Step 1: Identify the Factors of Competition**

List all factors on which the industry currently competes and invests. These
become the horizontal axis of the canvas.

**For a restaurant industry example:**
Price, Food Quality, Menu Variety, Ambiance, Service Speed, Wine Selection,
Location Convenience, Healthy Options, Brand Prestige

**Step 2: Score Each Competitor (1-5 scale)**

| Factor              | Competitor A | Competitor B | Industry Avg | YOUR Offering |
|---------------------|-------------|-------------|--------------|---------------|
| Price (low=high)    |             |             |              |               |
| Food Quality        |             |             |              |               |
| Menu Variety        |             |             |              |               |
| Ambiance            |             |             |              |               |
| Service Speed       |             |             |              |               |
| Wine Selection      |             |             |              |               |
| Location            |             |             |              |               |
| Healthy Options     |             |             |              |               |
| Brand Prestige      |             |             |              |               |

**Step 3: Plot the Value Curves**

```
High 5 |    *                          *
       |   / \                        / \
     4 |  /   \    *          *      /   \
       | /     \  / \        / \    /     \
     3 |/       \/   \      /   \  /       \
       |         \    *    /     \/         \
     2 |          \  / \  /                  \
       |           \/   \/                    \
Low  1 |            *                          *
       +------+------+------+------+------+------+
         Factor Factor Factor Factor Factor Factor
           1      2      3      4      5      6
```

**Reading the Canvas:**
- Convergent value curves = red ocean (everyone looks the same)
- Divergent value curve = potential blue ocean (you look different)
- A curve that zigzags between high and low = focused strategy
- A curve that is high everywhere = unfocused, high-cost strategy

### Three Qualities of a Good Blue Ocean Strategy (from the Canvas)

1. **Focus** — The value curve does not try to match competitors on all factors.
   Some factors are deliberately reduced or eliminated.
2. **Divergence** — The value curve stands apart from competitors' curves. If
   it mirrors others, you're in a red ocean.
3. **Compelling Tagline** — The strategy can be communicated in a single clear
   message that captures the value curve shape.

**Test:** If you cannot describe your strategy in a tagline that communicates
a divergent value curve, your strategy is likely unfocused or undifferentiated.

## 3. THE FOUR ACTIONS FRAMEWORK

The Four Actions Framework is the tool for creating a new value curve that
simultaneously pursues differentiation and low cost. It poses four questions
that challenge an industry's strategic logic.

### The Four Questions

```

```

### The ERRC Grid (Eliminate-Reduce-Raise-Create)

This is the working document that captures your Four Actions analysis.

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|
| Factors to completely remove     | Factors to increase well above   |
| from your offering               | the industry standard            |
|                                  |                                  |
| - [Factor 1]                     | - [Factor 1]                     |
| - [Factor 2]                     | - [Factor 2]                     |
| - [Factor 3]                     | - [Factor 3]                     |
|                                  |                                  |
|----------------------------------|----------------------------------|
| REDUCE                           | CREATE                           |
|----------------------------------|----------------------------------|
| Factors to reduce significantly  | Factors the industry has never   |
| below the industry standard      | offered — entirely new           |
|                                  |                                  |
| - [Factor 1]                     | - [Factor 1]                     |
| - [Factor 2]                     | - [Factor 2]                     |
| - [Factor 3]                     | - [Factor 3]                     |

### How ERRC Drives Value Innovation

```
COST REDUCTION                 VALUE INCREASE
     |                              |
     v                              v
 ELIMINATE ----> Removes cost   RAISE -----> Increases buyer
                 of factors                  value on factors
                 industry                    buyers care about
                 over-invests in             most
     |                              |
     v                              v
 REDUCE ------> Cuts cost by   CREATE ----> Opens entirely
                 reducing                    new sources of
                 over-served                 buyer value
                 factors
```

**Result:** Simultaneous cost reduction + value increase = VALUE INNOVATION.
This is the cornerstone of Blue Ocean Strategy. You do not choose between
differentiation and low cost — you achieve both.

### Decision Criteria for Each Action

**When to ELIMINATE a factor:**
- Customers don't actually value it (industry assumption, not customer reality)
- It primarily drives cost without driving buyer choice
- It exists because "we've always done it this way"
- Removing it would shock the industry but not the customer

**When to REDUCE a factor:**
- The industry has over-invested relative to customer needs
- Diminishing returns on further investment
- Customers would accept "good enough" on this factor
- Cost savings can be redirected to RAISE/CREATE factors

**When to RAISE a factor:**
- Customers consistently complain about industry-standard levels
- Higher performance on this factor drives switching behavior
- Competitors are under-investing relative to customer expectations
- Improvement is technically feasible at reasonable cost

**When to CREATE a factor:**
- Non-customers cite its absence as a reason for non-participation
- Adjacent industries offer it but your industry does not
- Technology has made it newly possible
- It addresses a job-to-be-done that current factors miss

## 4. REAL EXAMPLES — DETAILED ERRC ANALYSIS

### Example 1: CIRQUE DU SOLEIL

**Industry:** Traditional circus (Ringling Bros., Barnum & Bailey)

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|
| Star performers (high-cost acts) | Unique venue/tent experience     |
| Animal shows (ethical + cost)    | Artistic quality of performances |
| Aisle concession sales           |                                  |
| Multiple show arenas             |                                  |
|----------------------------------|----------------------------------|
| REDUCE                           | CREATE                           |
|----------------------------------|----------------------------------|
| Fun and humor (less slapstick)   | Theme/storyline for each show    |
| Thrill and danger emphasis       | Refined viewing environment      |
|                                  | Artistic music and dance         |
|                                  | Multiple unique productions      |
|                                  | Theater-level ticket pricing     |

**Result:** Cirque du Soleil eliminated the most expensive elements of circus
(animals, star performers) while creating theater-level experiences. Revenue per
show far exceeded traditional circuses. They didn't "win" against Ringling Bros. —
they made the comparison irrelevant by attracting theater-goers and corporate
clients, not traditional circus-goers.

**Non-customers captured:** Adults who would go to theater or concerts but
considered circus "for children." Corporate event buyers seeking unique experiences.

### Example 2: YELLOW TAIL WINE (Casella Wines)

**Industry:** U.S. wine market (dominated by premium/mid-tier wines)

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|
| Enological terminology on labels | Price relative to budget wines    |
| Aging qualities                  | Retail store involvement         |
| Above-the-line marketing         |                                  |
|----------------------------------|----------------------------------|
| REDUCE                           | CREATE                           |
|----------------------------------|----------------------------------|
| Wine complexity/range            | Easy drinking (sweet, smooth)    |
| Vineyard prestige                | Easy selection (2 types: red, white)|
| Wine tasting subtleties          | Fun, approachable brand identity |
|                                  | Kangaroo branding (memorable)    |

**Result:** Within 3 years, Yellow Tail became the #1 imported wine in the U.S.
by volume, surpassing French and Italian wines. It captured beer and cocktail
drinkers who found wine intimidating. Casella simplified everything: 2 varieties
(red and white), no confusing labels, sweet accessible taste.

**Non-customers captured:** Beer and cocktail drinkers (Tier 2 non-customers who
had actively rejected wine as too complex/pretentious).

### Example 3: SOUTHWEST AIRLINES

**Industry:** U.S. airline industry

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|
| First class / business class     | Friendly, fun service culture    |
| Meal service                     | Departure frequency              |
| Seat assignments                 | On-time reliability              |
| Hub-and-spoke connections        |                                  |
| Lounges                          |                                  |
|----------------------------------|----------------------------------|
| REDUCE                           | CREATE                           |
|----------------------------------|----------------------------------|
| Ticket prices (dramatically)     | Point-to-point routes            |
| In-flight amenities              | Speed of gate turnaround (15 min)|
| Baggage handling complexity      | Fun flying experience            |
|                                  | No-frills ticket flexibility     |

**Result:** Southwest created a new market between "fly" and "drive." By making
flying affordable and convenient for short routes, they pulled car travelers into
air travel. Their 15-minute gate turnaround (vs industry 45-60 minutes) enabled
higher aircraft utilization, supporting lower fares.

**Non-customers captured:** Car travelers on short routes (Tier 3 — had never
considered flying as an option for short trips).

### Example 4: NINTENDO Wii

**Industry:** Video game consoles (dominated by Sony PS3 and Microsoft Xbox 360)

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|
| Bleeding-edge graphics hardware  | Ease of use for non-gamers       |
| Hard drive storage               | Social/family gaming experience  |
| DVD/Blu-ray playback             |                                  |
|----------------------------------|----------------------------------|
| REDUCE                           | CREATE                           |
|----------------------------------|----------------------------------|
| Processing power                 | Motion-sensing controls          |
| Online multiplayer complexity    | Fitness/health gaming (Wii Fit)  |
| Game library for hardcore gamers | Inter-generational appeal        |
|                                  | Active physical gameplay         |

**Result:** The Wii outsold both PS3 and Xbox 360, despite being vastly inferior
in processing power and graphics. By eliminating the "arms race" on hardware specs,
Nintendo reduced manufacturing costs while creating entirely new value through
motion controls. Families, seniors, and non-gamers bought a console for the
first time.

**Non-customers captured:** Parents, seniors, fitness enthusiasts (Tier 3 —
people who had never considered buying a game console).

### Example 5: CURVES FITNESS

**Industry:** Fitness club industry

| ELIMINATE                        | RAISE                            |
|----------------------------------|----------------------------------|

worldwide. By targeting women who were intimidated by traditional gyms, they
captured a massive Tier 2 non-customer group (women who refused to go to gyms).

## 5. THE SIX PATHS FRAMEWORK

The Six Paths Framework provides systematic ways to look across conventional
industry boundaries and discover blue ocean opportunities. Each path reconstructs
market boundaries.

### Path 1: Look Across Alternative Industries

**Not competitors — alternatives.** What other industries solve the same problem
differently?

**Process:**
1. What is the job-to-be-done your customer is hiring a solution for?
2. What alternative industries address the same job?
3. Why do customers trade across these alternatives?
4. What factors make them switch between alternatives?

**Example:** Home Depot looked across professional contractors (expensive, skilled)
vs DIY stores (cheap materials, no guidance). Created guided DIY with in-store
expertise, capturing both professional-quality results and DIY pricing.

### Path 2: Look Across Strategic Groups Within the Industry

**Strategic groups** are companies within the same industry that pursue similar
strategies (e.g., luxury vs economy hotels).

**Process:**
1. Map strategic groups on two axes (typically price vs. value/scope)
2. Why do customers trade up to higher groups?
3. Why do customers trade down to lower groups?
4. What would capture both groups' buyers?

**Example:** Toyota Lexus — offered near-Mercedes quality at near-Toyota price
by eliminating prestige signaling costs and raising engineering quality.

### Path 3: Look Across the Chain of Buyers

**Three buyer roles:** Purchasers (who pay), Users (who use), Influencers
(who recommend). Industries tend to focus on just one.

**Process:**
1. Who is the purchaser, user, and influencer in your industry?
2. Which does the industry focus on?
3. What happens if you shift focus to a different buyer?

**Example:** Bloomberg shifted from selling to IT departments (purchasers) to
designing for traders (users), creating terminals traders loved. IT departments
bought what traders demanded.

### Path 4: Look Across Complementary Products and Services

**What happens before, during, and after your product is used?**

**Process:**
1. Map the total solution customers need (not just your product)
2. What pain points exist before, during, or after using your product?
3. Can you expand your offering to address these pain points?

**Example:** Dyson saw that vacuum cleaners required buying replacement bags
(complementary cost). Bagless cyclonic technology eliminated this pain point.

### Path 5: Look Across Functional-Emotional Orientation

**Some industries compete on function (rational, price-based). Others compete on
emotion (feelings, experience). Switch the orientation.**

**Process:**
1. Is your industry functionally or emotionally oriented?
2. What would happen if you added emotional value to a functional industry?
3. What would happen if you stripped emotion from an emotional industry?

**Example:** Swatch moved watches from functional (telling time) to emotional
(fashion accessory), creating a new market. QB House (Japan) moved haircuts from
emotional (salon experience) to functional (10-minute, $8 cut).

### Path 6: Look Across Time

**What trends are decisively shaping your industry? Project forward.**

**Process:**
1. What trends have a high probability of impacting your industry?
2. How will these trends change buyer value?
3. Can you offer that future value today?

**Example:** Apple's iTunes anticipated that digital distribution would replace
CDs. Instead of fighting piracy, Apple created a legal, convenient digital store.
Netflix anticipated streaming would replace physical media and pivoted years
before competitors.

### Six Paths Decision Matrix

For systematic application, score each path:

| Path                        | Opportunity Size (1-5) | Feasibility (1-5) | Time to Market (1-5) | Total |
|-----------------------------|------------------------|--------------------|-----------------------|-------|
| Alternative industries      |                        |                    |                       |       |
| Strategic groups            |                        |                    |                       |       |
| Chain of buyers             |                        |                    |                       |       |
| Complementary offerings     |                        |                    |                       |       |
| Functional-emotional switch |                        |                    |                       |       |
| Across time                 |                        |                    |                       |       |

Pursue the path(s) with the highest total score.

## 6. THE BUYER UTILITY MAP

The Buyer Utility Map systematically identifies where an industry currently
creates utility and where the biggest opportunity gaps exist.

### The Map Structure

```
               |  Customer  | Simplicity | Convenience | Risk      | Fun &    | Environmental |
               | Productivity|           |             | Reduction | Image    | Friendliness  |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Purchase     |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Delivery     |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Use          |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Supplements  |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Maintenance  |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
| Disposal     |            |           |             |           |          |               |
+--------------+------------+-----------+-------------+-----------+----------+---------------+
```

**Rows = Buyer Experience Cycle (6 stages):**
1. **Purchase** — How long does it take to find and buy the product?
2. **Delivery** — How is the product delivered to the buyer?
3. **Use** — What is the experience of using the product?
4. **Supplements** — What other products/services are needed alongside?
5. **Maintenance** — How easy is it to maintain and update?
6. **Disposal** — What happens when the product is no longer needed?

**Columns = Six Utility Levers:**
1. **Customer Productivity** — Does it save time, effort, or money?
2. **Simplicity** — Does it reduce complexity or mental effort?
3. **Convenience** — When and where can it be used?
4. **Risk Reduction** — Does it reduce financial, physical, or emotional risk?
5. **Fun & Image** — Does it provide enjoyment or social status?
6. **Environmental Friendliness** — Is it sustainable and eco-friendly?

### How to Use the Map

**Step 1:** Mark cells where the industry currently creates utility (fill with "X")
**Step 2:** Mark cells where your offering creates utility (fill with "O")
**Step 3:** Identify BLANK cells — these are untapped utility opportunities
**Step 4:** Prioritize blank cells using: customer pain severity (1-5) x
addressability (1-5)
**Step 5:** Design ERRC actions to fill the highest-priority blank cells

**AI Agent Advantage:** An AI agent can research each cell systematically,
gathering data on competitor offerings, customer complaints, and adjacent
industry solutions to populate the map with evidence rather than guesswork.

## 7. THREE TIERS OF NON-CUSTOMERS

Blue Ocean Strategy focuses not only on existing customers but on the vast ocean
of non-customers. There are three tiers, each requiring different approaches.

### Tier 1: "Soon-to-be" Non-customers

**Description:** People on the edge of the market who buy minimally out of
necessity but are mentally ready to leave. They use the industry's offering as
a stopgap while searching for something better.

**How to find them:** Look for customers with declining purchase frequency,
low engagement, or vocal complaints. Look for high churn rates.

**Example:** People who subscribed to a gym but go once a month. They haven't
canceled yet, but they're barely customers. Curves captured these by creating
a 30-minute, non-intimidating workout.

### Tier 2: "Refusing" Non-customers

**Description:** People who have consciously evaluated and rejected the
industry's offering. They know about it, considered it, and said "no."

**How to find them:** Survey people who considered buying but didn't. Analyze
competitor trial-to-conversion drop-off data. Study cancellation reasons.

**Example:** People who tried wine but found it too complex and pretentious.
Yellow Tail captured them with simple, sweet, fun-branded wine.

### Tier 3: "Unexplored" Non-customers

**Description:** People in distant markets who have never considered the
industry's offering as an option. They are the farthest from the current market.

**How to find them:** Look at what jobs-to-be-done your product could serve in
entirely different contexts. Study adjacent industries serving similar needs.

**Example:** Seniors who never considered buying a game console. Nintendo Wii
captured them with motion-controlled fitness and social games.

### Non-customer Analysis Framework

| Dimension               | Tier 1 (Edge)    | Tier 2 (Refusing) | Tier 3 (Unexplored) |
|--------------------------|------------------|--------------------|----------------------|
| Awareness of offering    | High             | High               | Low or none          |
| Reason for non-use       | Dissatisfaction  | Active rejection   | Never considered     |
| Size of opportunity      | Small-Medium     | Medium-Large       | Potentially massive  |
| Ease of capture          | Easiest          | Moderate           | Most difficult       |
| Research method          | Churn data       | Survey refusers    | Adjacent market study|
| Conversion approach      | Remove friction  | Address objections | Create new framing   |

## 8. BLUE OCEAN SHIFT — 5-STEP PROCESS

A structured process for moving from red to blue ocean (from "Blue Ocean Shift,"
2017).

### Step 1: Get Started

- Choose the right scope: a specific product-market to focus on
- Build a team with diverse perspectives (not just strategists)
- Define the current playing field and competitive boundaries

**AI Agent Execution:** Research the competitive landscape. Identify 3-5 key
competitors. Gather their pricing, positioning, and feature data.

### Step 2: Understand Where You Are Now

- Build the AS-IS Strategy Canvas showing current value curves
- Score all factors of competition for yourself and competitors
- Identify convergence zones where all players look alike

**AI Agent Execution:** Create the AS-IS Strategy Canvas using researched data.
Flag factors where all competitors score within 1 point of each other
(convergence = red ocean signal).

### Step 3: Imagine Where You Could Be

- Apply the Six Paths Framework (Section 5) to discover opportunities
- Map the Buyer Utility Map (Section 6) to find untapped spaces
- Analyze the Three Tiers of Non-customers (Section 7) to size opportunities

**AI Agent Execution:** Systematically apply all 6 paths. Score each using the
Six Paths Decision Matrix. Research non-customer tiers via market data.

### Step 4: Find How You Get There

- Build the ERRC Grid (Section 3) based on insights from Step 3
- Create 3-5 TO-BE Strategy Canvas options (new value curves)
- Evaluate each option against: buyer utility, price corridor, cost target

**AI Agent Execution:** Generate multiple ERRC Grid variations. Plot each as
a value curve on the Strategy Canvas. Score against the three evaluation criteria.

### Step 5: Make Your Move

- Select the strongest TO-BE Strategy Canvas
- Validate with rapid market tests (landing pages, prototypes, pilot markets)
- Build the full business model using the Business Model Canvas
- Plan the execution sequence: which ERRC actions first?

**AI Agent Execution:** Produce the final Strategy Canvas, ERRC Grid, and
completed Business Model Canvas. Create handoff to the Experiment Designer with
specific hypotheses to test.

## 9. COMMON TRAPS AND FAILURE MODES

### Trap 1: Blue-washing a Red Ocean Strategy

**Symptom:** Calling incremental improvements "blue ocean." Adding one new feature
while keeping everything else the same.

**Test:** Does your value curve DIVERGE from competitors? If it overlaps on 7
out of 10 factors, it's still red ocean.

**Fix:** Be bolder. Every blue ocean example involves ELIMINATING at least 2-3
factors the industry considers essential. If you haven't eliminated anything,
you haven't made a blue ocean move.

### Trap 2: All Create, No Eliminate

**Symptom:** Adding new factors without removing or reducing anything. This
creates a "Christmas tree" strategy — everything plus more.

**Test:** Is your cost structure higher than competitors? If yes, you haven't
achieved value innovation — you've just achieved differentiation at high cost.

**Fix:** For every factor you CREATE or RAISE, you must ELIMINATE or REDUCE at
least one factor. This is the discipline that makes blue ocean work.

### Trap 3: Technology-Driven Instead of Value-Driven

**Symptom:** Building blue ocean strategy around a technology rather than a
buyer value gap. "We have this cool technology, let's find a market."

**Test:** Did you start from buyer utility gaps, or from your technology
capabilities? Blue ocean is buyer-centric, not technology-centric.

**Fix:** Start with the Buyer Utility Map. Identify utility gaps first. Then
ask: what technology could fill these gaps?

### Trap 4: Ignoring the Price Corridor

**Symptom:** Creating great buyer value but pricing it out of reach for the
mass of target buyers.

**Test:** Is your price within the "price corridor of the mass"? Kim & Mauborgne
define this as the price band that captures the largest group of target buyers
when considering substitutes and alternatives (not just direct competitors).

**Fix:** Use the strategic pricing framework:
1. Identify the price of alternatives (not competitors — alternatives)
2. Find the "price corridor of the mass" (the range most target buyers accept)
3. Set your price within this corridor
4. Work backward to hit your cost target at this price

### Trap 5: Failing to Align the Organization

**Symptom:** Great strategy on paper but the organization can't or won't execute.

**Test:** Do employees understand the strategy? Do incentives align? Are the
right capabilities in place?

**Fix:** Apply "Fair Process" — Engagement (involve people in the decision),
Explanation (explain the reasoning behind the strategy), Expectation Clarity
(be clear about new roles and responsibilities).

### Trap 6: Copying Another Company's Blue Ocean

**Symptom:** Trying to replicate Southwest Airlines' or Cirque du Soleil's
specific ERRC grid in a different industry without understanding the underlying
logic.

**Test:** Did you derive your ERRC grid from YOUR industry's factors and YOUR
non-customers? Or did you copy someone else's grid?

**Fix:** Blue ocean thinking is a PROCESS, not a template. Every ERRC grid
must be derived from deep analysis of the specific industry context.

## 10. TIPPING POINT LEADERSHIP

For execution of a blue ocean move, Kim & Mauborgne propose Tipping Point
Leadership — the idea that change can happen fast when you concentrate resources
on the decisive factors.

### Four Organizational Hurdles

1. **Cognitive Hurdle** — People don't see the need for change
   - Fix: Make people experience the problem (ride-alongs, customer visits)
   - AI Agent support: Generate vivid competitive analysis showing trajectory

2. **Resource Hurdle** — Limited resources for transformation
   - Fix: Redirect resources from cold spots (low-impact activities) to hot spots
   - AI Agent support: Identify resource reallocation opportunities from ERRC grid

3. **Motivational Hurdle** — People aren't motivated to change
   - Fix: Focus on key influencers ("kingpins"), not mass persuasion
   - AI Agent support: Map stakeholder influence networks

4. **Political Hurdle** — Powerful vested interests resist change
   - Fix: Identify and neutralize opponents early, recruit powerful allies
   - AI Agent support: Model political risk and coalition-building scenarios

## 11. AI AGENT EXECUTION PROTOCOL

When applying Blue Ocean Strategy as an AI agent:

**Step 1: Industry Research (20% of effort)**
- Research 5+ competitors: pricing, features, positioning
- Identify industry factors of competition from reviews, comparisons, trade press
- Gather market size, growth rates, margin data

**Step 2: Strategy Canvas Construction (20% of effort)**
- List 8-12 factors of competition
- Score each competitor and industry average on all factors (1-5)
- Create the AS-IS Strategy Canvas
- Identify convergence zones and divergence opportunities

**Step 3: Blue Ocean Analysis (30% of effort)**
- Apply all Six Paths systematically
- Map the Buyer Utility Map
- Analyze three tiers of non-customers
- Score opportunities using the Six Paths Decision Matrix

**Step 4: ERRC Grid Design (20% of effort)**
- Generate 2-3 ERRC Grid options based on top-scoring paths
- Plot each as a TO-BE value curve on the Strategy Canvas
- Validate against: Focus, Divergence, Compelling Tagline
- Check price corridor of the mass

**Step 5: Integration and Output (10% of effort)**
- Select the strongest option
- Integrate into the Business Model Canvas
- Score key assumptions using the Risk Matrix
- Create handoff document for experiment design

### Integration with Business Model Canvas

The ERRC Grid directly feeds the Business Model Canvas:

```
ELIMINATE/REDUCE actions --> Reduce Cost Structure
                        --> Simplify Key Activities
                        --> Reduce Key Resources needed

RAISE/CREATE actions    --> Define Value Proposition
                        --> Identify new Customer Segments (non-customers)
                        --> Shape Revenue Streams (new pricing)
                        --> Determine new Channels (reaching non-customers)
```

## 12. QUICK-REFERENCE DECISION TREES

### "Should I apply Blue Ocean Strategy?"

```
Is your industry growing fast with high margins?
  YES --> Blue Ocean is optional, but can accelerate growth
  NO  --> Continue:

Are competitors' value curves converging?
  YES --> Strong Blue Ocean candidate
  NO  --> Continue:

Are there large groups of non-customers?
  YES --> Strong Blue Ocean candidate
  NO  --> Consider defensive strategy instead

Can you identify factors to ELIMINATE without losing core customers?
  YES --> Proceed with ERRC Grid
  NO  --> Start with Six Paths to find new angles
```

### "Which Six Path should I start with?"

```
Do you have adjacent industry knowledge?
  YES --> Path 1 (Alternative Industries)
  NO  --> Continue:

Are there clear strategic groups in your industry?
  YES --> Path 2 (Strategic Groups)
  NO  --> Continue:

Is your industry functionally or emotionally oriented?
  CLEAR ORIENTATION --> Path 5 (Switch it)
  MIXED --> Continue:

Are there major trends reshaping your industry?
  YES --> Path 6 (Across Time)
  NO  --> Paths 3 or 4 (Buyer Chain or Complements)
```
