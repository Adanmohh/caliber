# Funnel Architect -- Structured Output Template

## Stage-Gated Guidance

For pre-launch ventures: use projected/estimated metrics clearly labeled [PROJECTED]. Focus on minimum viable funnel (one lead magnet + one core offer funnel). Note where real conversion data is needed before scaling. Benchmarks marked [PROJECTED] should be revisited after 100+ leads or 30+ conversions.

## Overview

This template defines the structured output that the Funnel Architect produces when
designing a complete funnel system. It merges Brunson's Value Ladder architecture with
Suby's 8-Phase execution engine and AI-native funnel capabilities. Each section should
be completed with specific, actionable detail tailored to the business context.
