# Copywriter Output Template

## Phasing Guidance

**Phase 1 (MVP):** Big Idea + 3 headlines + landing page copy. Produce Phase 1 by default.
**Phase 2:** Email sequences + ad copy. Expand to Phase 2 when explicitly requested or when Phase 1 is validated.
**Phase 3:** Content calendar + full content system. Expand to Phase 3 when explicitly requested or when prior phases are validated.

Produce Phase 1 by default; expand to later phases when explicitly requested or when prior phases are validated.

## Stage-Gated Guidance

For pre-launch ventures: use projected/estimated metrics clearly labeled [PROJECTED]. Focus on minimum viable copy (Phase 1). Note where real conversion data and VOC data are needed before expanding. Use placeholder VOC from competitor reviews and market research until first-party data is available.

---

The complete structured deliverable format for the merged Copywriter. Every
engagement produces artifacts across six sections: VOC Mining Report, Message
Hierarchy, Copy Package, Content System Design, Copy Scorecard, and Content
Performance Dashboard.

Adapt section depth based on project scope. Use the Project Type Guide below
to determine which sections are required.

---

## Project Type Guide

```
PROJECT TYPE              REQUIRED SECTIONS
------------------------  -------------------------------------------
Full campaign             All 6 sections in full
Landing page              Sections 1-3 + Section 5 (Scorecard)
Email sequence            Sections 1-2 + Section 3 (emails only) + 5
Ad copy                   Sections 1-2 + Section 3 (ads only) + 5
Content strategy          Sections 1-2 + Section 4 + Section 6
Copy audit                Section 5 (Scorecard) + rewrite recommendations
Social media plan         Section 4 (Content System) + Section 6
Full copy + system        All 6 sections
```

## Quality Checklist: Before Delivering Any Output

```
RESEARCH QUALITY
  [ ] VOC data gathered from 3+ source types
  [ ] 50+ raw quotes minimum (100+ for major projects)
  [ ] All data categorized: Before/Dream/Objections
  [ ] Top messages ranked by frequency AND intensity
  [ ] Competitor messaging analyzed

COPY QUALITY
  [ ] Rule of One defined and applied throughout
  [ ] Primary framework selected and executed (PAS/AIDA/1-2-3-4)
  [ ] 5+ headline options with rationales
  [ ] All copy run through Seven Sweeps
  [ ] VOC language present throughout (not just in brief)
  [ ] Every claim passes "So What?" + "Prove It"
  [ ] Specificity check: no vague adjectives without proof
  [ ] All CTAs use first-person language
  [ ] Copy length matches audience awareness level
  [ ] 3+ A/B test recommendations with hypotheses

BRAND QUALITY (OGILVY)
  [ ] Big Idea identified and tested (Five-Question Test)
  [ ] Brand voice consistent across all copy elements
  [ ] Headlines follow Ogilvy principles (benefit, brand, specificity)
  [ ] Copy respects the reader's intelligence
  [ ] Every piece invests in brand personality

CONTENT SYSTEM QUALITY (GARY VEE)
  [ ] Pillar-to-micro mapping complete (30+ per pillar)
  [ ] Platform-native adaptation for each piece
  [ ] Jab/Hook ratio planned (minimum 3:1, target 5:1+)
  [ ] Content calendar with DTA slots
  [ ] Performance metrics defined
  [ ] Feedback loop process documented
```
