# Offer Designer -- Structured Output Template

## Purpose

This template defines the structured output the Offer Designer produces when auditing,
redesigning, or creating an offer using Alex Hormozi's Grand Slam Offer framework and Value
Equation from "$100M Offers." Every section must be completed. Every recommendation must be
grounded in a specific framework element. Every score must be justified.

---

## Stage-Gated Guidance

For pre-launch ventures: use projected/estimated metrics clearly labeled [PROJECTED]. Focus on minimum viable offer. Note where real conversion data is needed before scaling. Prioritize Sections 1-8 (core offer design) before competitive analysis and pricing optimization, which benefit from real market feedback.

## When to Use This Template

Produce this output when:
- Auditing an existing offer for weaknesses and improvement opportunities
- Redesigning an offer to achieve Grand Slam status
- Creating a new offer from scratch
- Comparing offer variations to identify the strongest design
- Preparing handoff documents for copywriting or funnel implementation agents

---

#### 3.2 Value Score Calculation

```
VALUE SCORE CALCULATION
=======================

Formula: (Dream Outcome x Perceived Likelihood) / (Time Delay x Effort & Sacrifice)

Note: Time Delay and Effort scores use inverted scale (10 = best/minimal).
The formula uses the scores directly, meaning higher denominator scores
(less delay/effort) produce a smaller denominator value and thus a different
ratio interpretation. Use the raw scores alongside the composite.

Current:  ([DO] x [PL]) / ([TD] x [ES]) = [Numerator] / [Denominator] = [Score]

Rating: [See interpretation table]

  < 1.0   = Critically weak -- major redesign needed
  1.0-2.0 = Below average -- significant gaps
  2.0-4.0 = Average -- competitive but not compelling
  4.0-8.0 = Strong -- differentiated and attractive
  8.0-15.0 = Excellent -- people feel compelled to buy
  15.0+    = Grand Slam territory
```
