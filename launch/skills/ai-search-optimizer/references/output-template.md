# AI Search Optimizer Output Template: AI Search Optimization Deliverable

> This template defines the structured deliverable that the AI Search Optimizer produces for each AI search optimization engagement. It combines entity SEO, LLM visibility, EEAT assessment, and competitive analysis into a comprehensive, scored, and actionable AI search optimization plan.

---

## Pre-Launch Venture Guidance

For pre-launch ventures: Focus on foundation-building over optimization. Use projected metrics labeled [PROJECTED]. Prioritize organic/low-budget channels. Note where live data is needed before scaling.

## How to Use This Template

For each project, fill out every section below. Sections marked [REQUIRED] must always be completed. Sections marked [CONDITIONAL] are included when relevant to the project scope. Every recommendation must include: rationale, expected impact, implementation guidance, and success metric.

### Quality Checklist (Before Delivery)

- [ ] All scoring systems are populated with actual scores, not placeholders
- [ ] Every recommendation has a specific action, owner, and timeline
- [ ] AI visibility audit tested on at least 3 AI platforms with 30+ queries
- [ ] Entity assessment covers all platforms in the sameAs audit
- [ ] EEAT analysis includes specific evidence for each signal score
- [ ] Competitive analysis covers at least 3 direct competitors
- [ ] Roadmap actions are prioritized by impact and sequenced realistically
- [ ] All schema recommendations include ready-to-implement JSON-LD code
- [ ] Monitoring protocol is defined with specific cadence and tools
- [ ] Document reviewed for accuracy of all statistics and claims

## Appendix C: Glossary

| Term | Definition |
|---|---|
| AEO | Answer Engine Optimization -- optimizing for AI answer engines |
| GEO | Generative Engine Optimization -- optimizing for generative AI search |
| RAG | Retrieval-Augmented Generation -- how AI models search the web and cite sources in real-time |
| EEAT | Experience, Expertise, Authoritativeness, Trustworthiness |
| Entity | A unique, recognizable thing (person, brand, concept, place) with defined attributes |
| Knowledge Graph | Google's database of entities and their relationships (500B+ facts, 5B+ entities) |
| Knowledge Panel | The information box for recognized entities in Google search results |
| LLM | Large Language Model (e.g., GPT, Gemini, Claude) |
| sameAs | Schema.org property linking an entity to its profiles on other authoritative platforms |
| AI Overview | Google's AI-generated answer at the top of search results (2B+ monthly users) |
| Citation | When an AI system references and attributes information to a specific source |
| Topical Authority | Recognition as the comprehensive expert source on a topic |
| llms.txt | Emerging standard for guiding AI systems on how to use website content |
| Zero-click | When a user gets their answer from AI without clicking to any website |
| Entity Home | The definitive page on your site that describes what your entity IS |
| Answer Chunk | A discrete block of text after an H2/H3 that directly answers a potential query |
| Fan-out Query | Google's process of issuing sub-queries to find sources for AI Overviews |
| Parametric Knowledge | Information an LLM "knows" from training data (no citation generated) |
| Entity Confidence Score | How certain an AI system is that it has correctly identified an entity |
| Brand Search Volume | Monthly search volume for your brand name -- strongest predictor of AI citations (0.334 correlation) |
